# 🍷 Veloura – Luxury Restaurant Website

Veloura is a modern, elegant, and fully responsive luxury restaurant website designed to deliver a premium dining experience through a clean and stylish interface. It focuses on aesthetics, smooth user interaction, and a refined UI suitable for high-end restaurant branding.

---

## ✨ Features

- Luxury and elegant UI design  
- Fully responsive layout (mobile, tablet, desktop)  
- Single-page smooth scrolling experience  
- Interactive JavaScript AI chatbot  
- Reservation button with smooth confirmation message  
- Clean animations and hover effects  
- Minimal and modern user interface  

---

- Home

- Contact / Reservation
